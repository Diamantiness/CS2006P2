inputValidator¶

inputValidator(n, alpha)¶

Validates the modulus and multiplier for an IntricateInteger.

Parameters:

  • n – (int) The modulus.

  • alpha – (int) The multiplier.

Returns:

(bool) True if the input is valid.

Raises:

ValueError – If modulus or multiplier is invalid.
